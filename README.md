#### Dependências 
Go present tool

Com Go 1.16: `go install  golang.org/x/tools/cmd/present@latest`

### Como utilizar 
Para ver a apresentação digite na altura do diretório raiz `present` e acesse o endereço exibido.
(provavelmente será http://127.0.0.1:3999)
